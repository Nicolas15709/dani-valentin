# Instrucciones para agregar la música

Para que la música de fondo funcione correctamente, necesitas agregar el archivo de audio:

1. Descarga la canción "A mi me vuelve loco tu forma de ser"
2. Convierte el archivo a formato MP3 si no lo está
3. Renombra el archivo a: `a-mi-me-vuelve-loco.mp3`
4. Coloca el archivo en esta carpeta: `public/music/`

## Alternativa: Usar YouTube

Si prefieres usar un enlace de YouTube, puedes:
1. Usar un servicio como https://ytmp3.nu/ para descargar el audio
2. O modificar el componente BackgroundMusic para usar un reproductor de YouTube embebido

## Nota
El componente BackgroundMusic ya está configurado para buscar el archivo en:
`/music/a-mi-me-vuelve-loco.mp3`
